🌊 앱 개요
💡 이 앱은 무엇인가요?

하루를 설계하고 기록하는 개인 라이프 플래너예요. 플래너(My Day), 영어 스피킹(Level Up), 날씨·외출 가이드(Set Go) 세 가지 도구가 하나의 앱으로 통합돼 있어요. Claude AI와 함께 대화하며 만들었어요.

⛵
My Day
일일 플래너 + 가계부
할일 · 이벤트 · 감사일기 · 지출입력 · 영수증스캔
🌱
Level Up
OPIc 영어 스피킹 코치
일일학습 · 스크립트생성 · 롤플레이 · 노트
🌊
Set Go
날씨 기반 외출 가이드
날씨 · 옷차림 추천 · 오늘의 팁 · 지역 설정

⚡ 앱 동작 흐름
📱
Step 1
앱 접속
브라우저에서 this-time-next-year.vercel.app 접속 → Vercel 서버가 index.html 전송 → 브라우저에서 React 앱 실행
🏠
Step 2
홈 화면 표시
바다 배경 홈화면 표시 → ⛵ My Day / 🌱 Level Up / 🌊 Set Go 버튼 선택
💾
Step 3
데이터 저장·불러오기
할일·가계부·OPIc 학습 기록은 모두 폰의 localStorage에 저장 → 앱을 껐다 켜도 유지됨 → 다른 사람 폰에서는 안 보임
🤖
Step 4
AI 기능 사용
오늘의 질문 · OPIc 스크립트 생성 · 날씨 정보 · 하루 총평 → 앱이 Vercel 서버에 요청 → 서버가 Gemini AI에 전달 → 결과 반환
✅
Step 5
결과 표시
AI가 생성한 텍스트를 앱 화면에 표시
🤖 AI 기능 동작 방식
1
사용자가 앱에서 AI 기능 실행 (질문 생성 · 스크립트 · 날씨 등)
2
앱이 POST /api/gemini 요청 전송 (프롬프트 포함)
3
Vercel 서버의 api/gemini.js 가 요청 수신 (API 키는 여기에 안전하게 보관)
4
서버가 Google Gemini API 에 요청 전달
5
Gemini가 응답 생성 → 서버 → 앱 → 화면에 표시

💾 데이터 저장 구조

모든 데이터는 내 폰의 localStorage에 저장돼요. 서버에는 아무것도 안 올라가요.
